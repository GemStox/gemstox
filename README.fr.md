## What is GemStox?

GemStox est un moteur de signaux boursiers mathématiquement prouvé. Transforme les données brutes du marché en signaux clairs avec un filtre de bruit de 99%. Aucun diplôme en finance requis.

- Plusieurs classes de stratégie — momentum, valeur, breakout, mean-reversion
- Filtre de bruit à 99% — supprime les alertes inutiles, ne montre que les opportunités probables
- Timing clair — entrée, stop-loss et take-profit pour chaque signal
- Dimensionnement Kelly — taille de position mathématiquement optimale
- Validé par cross-validation — chaque stratégie testée sur données hors échantillon
