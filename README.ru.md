GemStox — математически доказанный движок биржевых сигналов. Преобразует сырые рыночные данные в чёткие сигналы на покупку/продажу с фильтрацией шума 99%. Финансовое образование не требуется.

- Несколько классов стратегий — моментум, стоимость, пробой, возврат к среднему
- 99% фильтр шума — отсеивает бесполезные оповещения, показывает только высоковероятные сигналы
- Чёткие сигналы — вход, стоп-лосс и тейк-профит для каждого сигнала
- Позиционирование по критерию Келли — математически оптимальный размер позиции
- Кросс-валидация — каждая стратегия проверена на вневыборочных данных
