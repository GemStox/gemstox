## 🚀 What is GemStox?

GemStox is an AI-powered stock scanner that helps traders find high-probability opportunities across **US, European, and Asian markets** in real-time. Built on the TradingAgents multi-agent LLM framework.

### Key Features
- **Real-time scanning** — 10,000+ stocks across global markets
- **AI ensemble signals** — Multi-agent analysis combining fundamentals, sentiment, and technicals
- **19 languages** — Full localization including Chinese, Japanese, Korean, Arabic, Russian
- **Kelly Criterion position sizing** — Math-backed risk management
- **14-day free trial** — No credit card required
