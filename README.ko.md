## What is GemStox?

GemStox는 수학적으로 입증된 주식 신호 엔진입니다. 99% 잡음 필터로 원시 시장 데이터를 명확한 매수/매도 신호로 변환합니다. 금융 학위가 필요 없습니다.

- 여러 전략 클래스 — 모멘텀, 가치, 브레이크아웃, 평균 회귀
- 99% 잡음 필터 — 쓸모없는 알림을 제거하고 높은 확률의 설정만 표시
- 명확한 매매 타이밍 — 각 신호에 진입가, 손절가, 목표가 표시
- 켈리 기준 최적 포지션 크기
- 교차 검증 완료 — 모든 전략이 샘플 외 데이터로 검증됨
