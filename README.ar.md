## What is GemStox?

GemStox هو محرك إشارات أسهم مثبت رياضياً. يحول بيانات السوق الخام إلى إشارات بيع وشراء واضحة مع معدل تصفية ضوضاء 99%. لا يتطلب شهادة في المالية.

- فئات استراتيجية متعددة — الزخم، القيمة، الاختراق، العودة إلى المتوسط
- تصفية ضوضاء بنسبة 99% — إلغاء التنبيهات غير المفيدة، عرض الفرص عالية الاحتمال فقط
- توقيت واضح للبيع والشراء — دخول، وقف خسارة، وجني أرباح لكل إشارة
- تحديد حجم المركز بمعيار كيلي — الحجم الأمثل رياضياً لكل صفقة
- تم التحقق بالتحقق المتبادل — كل استراتيجية مختبرة على بيانات خارج العينة
