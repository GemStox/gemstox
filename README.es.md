## What is GemStox?

GemStox es un motor de señales bursátiles matemáticamente probado. Convierte datos brutos del mercado en señales claras con un filtro de ruido del 99%. Sin necesidad de título en finanzas.

- Múltiples clases de estrategia — momentum, valor, breakout, mean-reversion
- Filtro de ruido del 99% — elimina alertas inútiles, muestra solo configuraciones probables
- Timing claro — entrada, stop-loss y take-profit en cada señal
- Dimensionamiento Kelly — tamaño de posición matemáticamente óptimo
- Validación cruzada — cada estrategia probada con datos fuera de muestra
